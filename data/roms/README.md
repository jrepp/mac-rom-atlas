# ROM Inputs

Stable local location for ROM files used by `mac-rom-atlas`.

Raw ROM images are intentionally ignored in this public repository. Project metadata should refer to ROMs by repo-relative paths under this directory plus hashes and descriptive names.

Current ROMs referenced by the SuperMario analysis project:

- `data/roms/old-world/2mb/1994-09 - 5BF10FD1 - Quadra 660av & 840av.ROM`
- `data/roms/old-world/2mb/1994-05 - B6909089 - PowerBook 520 520c 540 540c.ROM`
- `data/roms/old-world/2mb/1995-08 - 4D27039C - Powerbook 190 & 190cs.ROM`
- `data/roms/basiliskii/Quadra-900.rom`
